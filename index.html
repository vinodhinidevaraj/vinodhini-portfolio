Amazon Customer Behavior Analysis

Conducted end-to-end exploratory and statistical analysis on Amazon Customer Survey data (602 observations). Used Python, statistical testing, regression modelling, and clustering techniques to uncover insights into customer satisfaction, purchase behaviour, and demographic trends.

  • Tools Used: Python, Pandas, Matplotlib, Scikit-learn
  • Key Insight: Rating Accuracy strongly impacts Shopping Satisfaction (r = 0.514)
  • Model: Linear Regression (R² = 0.264, p < 0.001)
  • Segmentation: K-Means clustering identified customer behaviour groups
